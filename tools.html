JWT Decoder & Inspector
auth · jwt · base64
Available

Decode JWT tokens and inspect headers, claims, and signature structure. Flags common weaknesses: alg:none, algorithm confusion risks, missing exp / iat, and suspicious kid patterns.

Findings
⚠ No exp claim    ⚠ HS256 with short secret

Password Entropy Checker
auth · entropy · bruteforce
Available

Estimate password strength in bits of entropy against dictionary, mask-based, and hybrid attacks. Assumes worst-case attacker with GPU cluster and leaked-password corpus. No password is stored or transmitted.

Summer2024!

Hash Analyzer
crypto · hash · identification
Available

Identify hash algorithms from format and length. Recognizes MD5, SHA-1, SHA-256, bcrypt, scrypt, NTLM, Kerberos TGS, and common web app formats. Flags deprecated algorithms and password storage anti-patterns.

Input (example)
$2b$12$N9qo8uLOickgx2ZMRZoMye...
Identified
bcrypt · cost=12 · 60 chars
Assessment
✓ Modern password hash, reasonable cost factor

Log Pattern Parser
logs · detection · iocs
Available

Paste raw log output to extract IPs, domains, file hashes, user agents, and URL paths. Flags suspicious patterns: path traversal attempts, SQL injection fingerprints, and known LOLBin invocation strings.

Input (example)

Extracted
IP: 10.0.4.12 · Path: /api/users · Status: 500
Findings
⚠ SQL injection pattern in query string
