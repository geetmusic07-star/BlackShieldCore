Architecture notes
Every module is a static HTML page sharing a single shared.css stylesheet. There is no application shell, no SPA router, and no client-side state management — each route is a fully pre-rendered document. Inputs to client-side tools (JWT, hash analyzer, log parser) stay in browser memory and are never transmitted. The site works with JavaScript disabled for all read-only content; tools require JS to run.
