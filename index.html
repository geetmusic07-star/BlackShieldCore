Financial Services

Bypassing 2FA via Race Condition

Analysis: Identified a severe multi-threading flaw in the OTP validation microservice of a major payment gateway. Simultaneous parallel requests exhausted and bypassed rate-limiting counters during the token lifecycle phase.

Remediation: Implemented strict database row-level locking and atomic Redis operations during token validation. Deployed canary monitoring across all OTP endpoints.

$2.1M+ in potential fraud prevented. Patched globally within 72 hours.

Enterprise SaaS

LLM Prompt Injection Shield

Problem: Enterprise AI agents were vulnerable to adversarial prompts that could chain tool-use permissions into unauthorized data access, resulting in customer PII leakage.

Solution: Developed an intermediary neural filter trained on 50K+ adversarial payloads. Deployed as an inline proxy with sub-15ms overhead. Self-healing via continuous red-team feedback loop.

99.4% detection rate. Zero false positives in 18-day production run.
